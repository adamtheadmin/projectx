# Project X

This is a Space Shooter video game that I made a few years ago as an example. This project uses:

  - Node Webkit
  - jQuery

To Start:
  - CD into the repo after cloning it
  - Install NW.js with `sudo npm install -g nw`
  - Run the application `nw`
